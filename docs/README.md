# Documentation Directory

This directory contains all documentation for the LeadSpark application.

## Directory Structure

- **setup/**: Contains guides for setting up various components of the application
  - `EMAIL_SETUP_GUIDE.md`: Guide for setting up email functionality

- **features/**: Contains documentation for application features
  - `CONTACT_ENRICHMENT.md`: Documentation for the contact enrichment feature

- **deployment/**: Contains guides for deploying the application and its features
  - `DEPLOYMENT_GUIDE.md`: General deployment guide
  - `CONTACT_ENRICHMENT_DEPLOYMENT.md`: Guide for deploying the contact enrichment feature
  - `FIELD_TRACKING_DEPLOYMENT.md`: Guide for deploying field tracking enhancements

## Using This Documentation

- Start with the setup guides if you're setting up the application for the first time
- Refer to the feature documentation to understand how specific features work
- Use the deployment guides when deploying new features or updates

## Contributing to Documentation

When adding new documentation:

1. Place it in the appropriate subdirectory based on its purpose
2. Follow the existing naming convention (ALL_CAPS_WITH_UNDERSCORES.md)
3. Include a clear title and description at the top of the document
4. Use markdown formatting for readability
